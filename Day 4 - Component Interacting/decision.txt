# Render giao diện khác nhau dựa vào Props
- Sử dụng props để đưa ra quyết định
- Ví dụ:
function LoginMsg(props) {
  if (props.password === 'a-tough-password') {
    return <h2>Sign In Successful.</h2>
  } else {
    return <h2>Sign In Failed..</h2>
  }
}
- Trong ví dụ trên, so sánh giá trị thuộc tính của props - password có bằng a tough password không để đưa ra quyết định
