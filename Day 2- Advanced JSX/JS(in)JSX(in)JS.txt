# Nếu như muốn tránh việc đọc code như JSX, cần sử dụng dấu ngoặc nhọn để đánh dấu rằng: Đây là code JS chứ không phải JSX
- Ví dụ:
        root.render(<h1>2 + 3</h1>);
        => Màn hình hiển thị 2 + 3
        root.render(<h1>{2 + 3}</h1>);
        => Màn hình hiển thị 5

- Bản thân dấu ngoặc nhọn sẽ không được coi là JSX hay JavaScript. Chúng là các dấu hiệu báo hiệu sự bắt đầu và kết thúc của việc chèn JavaScript vào JSX, tương tự như dấu ngoặc kép báo hiệu ranh giới của một chuỗi.
